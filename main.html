Question-1: What are the different ways to select an element in the DOM?

There are five different ways to select an elementin the DOM. They are: 1.getElementsByTagName, 2.getElementById, 3.getElemenstByClassName, 4.querySelector and 5.querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML sets and gets HTML content. innerText sets and gets only visible text and textContent sets and gets all text including hidden and visible.

Question-3: What is event delegation in the DOM?

Event delegation is a method where child elements can be handled by attachingan event listener to the parent element.

Question-4: What is event bubbling in the DOM?

Event bubbling is a propagation of the DOM, whrer when an event occurs on a child element, it rises up to the parent element.
